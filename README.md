# AI Resume Intelligence Platform

AI-powered Resume Analysis Platform built using Python, Streamlit, SQLAlchemy, NLP and LLM-based Skill Extraction.

## Features

- User Authentication
- Resume Upload (PDF/DOCX)
- Job Description Upload
- Skill Extraction
- Resume-JD Matching
- Match Score Calculation
- Missing Skill Detection
- Learning Recommendations
- Resume Library
- Dashboard Analytics

## Tech Stack

- Python
- Streamlit
- SQLAlchemy
- SQLite
- NLP
- LLM Integration
